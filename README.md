Packer Debian
=============

### Packer config for Debian and Virtualbox

- Debian Stretch 9.1
- Rbenv
- Bundler
- Passenger 5
- Ruby 2.2.2
- Ruby 2.2.4 (default)

### Requirements

- Packer : https://www.packer.io/downloads.html

### Usage

Build a new box
```
$ bash build
```

Test box
```
$ vagrant up
```
